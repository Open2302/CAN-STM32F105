Editing code

CoIDE integrates CDT Editor which is a part of CDT project of Eclipse. This section describes how to edit code in CoIDE, if you want to get more help, you can visit eclipse cdt project home.

To edit code using CoIDE, you could perform as follows:

 
  1. 1. Select File > New File.
  2. 2. Type file name for the new file and then click Finish button.
  3. 3. Edit your own application code.
  4. 4. If necessary, right click at the left edge of editor, select Show Line Numbers to enable the function or disable.
  5. 5. If necessary, right click at the left edge of editor, select Folding > Enable Folding to enable the function or disable.
  6. 6. If necessary, move cursor to a code field then right click, select Open Declaration to open a declaration file.
