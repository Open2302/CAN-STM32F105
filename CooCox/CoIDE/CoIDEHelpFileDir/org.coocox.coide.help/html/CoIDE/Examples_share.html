Sharing example

To share examples using CoIDE, you just need to perform the following steps:

 
  1. 1. If you have not login, you must login first.
  2. 2. If you don not have a account, click the link named "Get new CooCox account?" at the bottom of Login Dialog.
  3. 3. Type the Title of your code snippet.
  4. 4. Click Edit in the editor.
  5. 5. Edit you example code and click the submit button.
  6. 6. View your example code snippet.
