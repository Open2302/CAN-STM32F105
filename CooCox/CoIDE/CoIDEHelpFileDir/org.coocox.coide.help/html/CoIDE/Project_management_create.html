Creating project

To create an embedded project using CoIDE, you only need to perform the following steps:

 
  1. 1. Select Project > New Project.
  2. 2. Type a name for your project. For example, in the Project name field, type "MyFirstProject".
  3. 3. Select a manufacturer. For example, choose 'Nuvoton'.
  4. 4. Select a chip. For example, choose 'NUC140VE3CN'.
  5. 5. Check a component. For example, select 'GPIO'. CoIDE will automatically check the components which the checked component depends on.
  6. 6. You can review the result in Component View.
  7. 7. Click Drivers to view the driver pages.
  8. 8. Check a component. For example, select 'HD44780'. CoIDE will automatically check the components which the checked component depends on.
  9. 9. Finish creating project and you can review the result.
