Peripherals View（CoAssistant）

The Peripherals View can be used to view and Modify the vaule of Peripherals' registers.Values that have changed are highlighted in the Peripherals view when your program stops.

To use CoAssitant in CoIDE, you only need to perform the following steps:

 
  1. 1. Click Peripherals under the View menu.
  2. 2. CoIDE will open the Peipherals View.
  3. 3. Click the name of Peripheral to view the base address and description for this peripheral.
  4. 4. Click "+" to show the register list and view the value of these registers.
  5. 5. The Register that have changed are highlighted in in the Peripherals view.
  6. 6. Click the name of Register to open the detail information view.
  7. 7. In this view, you can get the description, RW access, address offset and the reset value of the register.
  8. 8. You can modify the value of register when you debug a project.
