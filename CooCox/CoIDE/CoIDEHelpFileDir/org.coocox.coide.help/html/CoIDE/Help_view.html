Help Contents

You can get CoIDE's version information from the Help menu, also you can get its User guide from this menu.

 
  1. 1. You can open the help contents by clicking Help Contents under the Help menu.
  2. 2. You fill find help contents shown in a new window.
  3. 3. The Content view shows the topic of the Help Content.
  4. 4. Open the Index view to search the topic.
  5. 5. Open the Search Results view to search a string in the content.
  6. 6. Open the Bookmarks view to manage your bookmark.
