CooCox CoIDE V1.4.1      4/1/2012

  • [CooCox CoIDE-Improved]
    • Added: Support Cortex-M4
    • Added: Support ARM Embeded GCC (https://launchpad.net/gcc-arm-embedded)
    • Added: Driver components page in Component Repository view
    • Added: 19 driver components based on CoX Peripheral Interface
    • Added: Support ST STM32F4 series and Holtek HT32F1755/2755 series
