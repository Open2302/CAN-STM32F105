Opening an existing project

To open an existing project using CoIDE, you only need to perform the following steps:

 
  1. 1. If necessary, you can select Project > Close Project to close the current project.
  2. 2. After closing the project, all the windows are clean.
  3. 3. Select Project > Open Project....
  4. 4. To open a CoIDE project, locate the directory of the project file with suffix of '.cob'.
  5. 5. Check your project.
  6. 6. To open a MDK project, locate the directory of the project file with suffix of '. uvproj'.
  7. 7. You can specify the directory of CoIDE project converted from the MDK project.
  8. 8. View folders and specify the directory.
  9. 9. Click OK.
  10. 10. Check your project.
