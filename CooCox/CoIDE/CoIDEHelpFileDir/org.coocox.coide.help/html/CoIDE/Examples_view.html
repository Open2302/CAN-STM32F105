Viewing example

To view examples using CoIDE, you just need to perform the following steps:

 
  1. 1. Click the component which shows "with x example(s)" in the Component View of CoIDE.
  2. 2. The example view will appear and display all the code snippets of the selected component.
  3. 3. Click view link of the example you are interested.
  4. 4. The code snippet will display in the code editor viewer.
  5. 5. You can click "More Example for xxx" to view more example for the selected chip.
  6. 6. Other example will show in more example view based on the component classification.
  7. 7. Click the example name can open the example in the edit view.
